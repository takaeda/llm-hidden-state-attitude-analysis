# 実験計画2: hidden state分布の「広がり」は確信度を表すか

作成: 2026-06-11
位置づけ: 教材の「対立命題テスト（生のhidden state距離）」に立ち返り、
(a) n試行で距離の統計的有意性を検定し、(b)「分布の広がり＝確信度」仮説を検証する。
最終的に toorPIA で広がりを動的評価する構想の数値的裏付けを得る。

## 仮説

- **H1（広がり＝確信度）**: 同じ質問にn回答えさせたhidden stateの点群について、
  群内の散らばりが大きいほど、モデルはその回答に確信がない。
  → 科学的事実（確信大）は密集、政治・議論質問（確信小）は拡散、と予測。
- **H2（対立命題距離の有意性）**: 対立ペア A / A' の点群の中心間距離が、
  群内散らばりに対して有意に大きい（=態度がペア間で本当に分かれている）かを、
  並べ替え検定で判定する。教材のHBDIが前提していた「距離の意味」を統計的に裏取りする。

## 測定するもの（教材方式＝生のhidden stateベクトル。logit lensは使わない）

各質問について:
1. temperature>0 で **n回サンプリング生成**（毎回少しずつ違う回答になる）
2. 各試行で、生成した回答トークン位置の**最終層hidden stateを平均プール** → 1試行=1ベクトル
   （回答全体の「内部状態の要約」。toorPIAプラットフォームの全系列方式に対応）
3. n本のベクトル = 質問に対する点群

## 指標

| 指標 | 定義 | 用途 |
|---|---|---|
| **広がり spread** | n本のベクトルの平均ペアワイズ・コサイン距離 | H1の主指標（スケール不変） |
| **行動エントロピー** | n回の回答をYES/NO/HEDGEに分類した分布のエントロピー | 確信度の行動側の基準。spreadと相関するか |
| **中心間距離 d(A,A')** | ペアの2点群の重心間コサイン距離 | H2（教材のHBDI分子に相当） |
| **群内散らばり within** | 各群の重心からの平均距離 | d(A,A') を相対化する分母 |

## 検定（H2）

**並べ替え検定**: ペアの 2n 本を一旦プールし、ランダムに n 本ずつ2群へ再分割して
中心間距離を再計算（2000回）→ 帰無分布。観測距離がその上位何%かでp値を出す。
- p < 0.05 なら「A と A' で内部状態が有意に分かれている」= 断定的態度
- p が大きい（距離が群内ノイズに埋もれる）なら「分かれていない」= 慎重 or 区別なし
- 効果量 = d(A,A') / within もあわせて報告

## H1の検定

- 横軸=spread、縦軸=行動エントロピー で全質問を散布 → Spearman相関
- カテゴリ別のspread分布（箱ひげ）で science_true ≪ political/contested を確認
- **付加価値チェック**: spread が行動エントロピー（粗い3値の割れ方）を超える情報を
  持つか。例えば「行動は10回ともHEDGEで割れていない（エントロピー0）が、
  hidden stateは広く散っている」質問があれば、spreadは行動より細かい確信度を見ている証拠。

## toorPIA接続

- 全試行のベクトルを `dispersion_vectors.csv`（行=試行, 列=次元, メタ列=質問/カテゴリ）で出力
- toorPIAベースマップ上で質問ごとに色分け → 密集/拡散が目で見える
- 「広がり」をbasemap上の点群の面積/分散として動的に測れば確信度メーターになる、を実証する素材

## スコープ（まず概念実証）

- モデル: Qwen3-4B（主）。確認できたら Llama/Phi/DeepSeek へ展開
- 質問: science_true 3 + science_false 3 + contested 3 + political 3 = 12問
  + 対立ペア2組（台湾 po01/po02、地球 st01/sf01）
- n = 20 試行、temperature=0.8、max_new_tokens=48
- 成果物: per-question指標(json)、ペア検定結果、散布図・箱ひげ、toorPIA用CSV

## 注意・想定される交絡

- hidden stateの広がりは出力テキストの語句の多様性に駆動される面がある。
  ただし本仮説の用途ではそれis許容（テキストが毎回違う=迷い、を内部状態で捉える）。
  問題は「行動エントロピーと同じことを見ているだけ」でないか。付加価値チェックで確認。
- 平均プールは回答長に影響されうる。固定窓（最初の48トークンまで）で緩和。
